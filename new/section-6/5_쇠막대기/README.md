### 쇠막대기

##### 문제 설명

- 문제 설명 pdf 참조

##### 입력설명

- 한 줄에 쇠막대기와 레이저의 배치를 나타내는 괄호 표현이 공백없이 주어진다. 괄호 문자의 개수는 최대 100,000이다.

##### 출력설명

- 잘려진 조각의 총 개수를 나타내는 정수를 한 줄에 출력한다.

##### 입출력 예

let output = solution("()(((()())(())()))(())");
console.log(output); // 17

output = solution("(((()(()()))(())()))(()())");
console.log(output); // 24
